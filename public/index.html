HTML Parser API

A powerful REST API service for extracting content from web pages using CSS selectors

API Endpoints

POST /api/parse

Extract content from a webpage using CSS selectors.

{ "parseURL": "https://example.com", "selector": ".content p", "method": "text", "extra": "href" }

GET /api/logs

Get recent request logs (for monitoring/debugging).

GET /api/health

Health check endpoint.

Methods

  • text - Extract text content
  • html - Extract HTML content with tags
  • attribute - Extract specific attributes (requires 'extra' parameter)

Example Usage

curl -X POST /api/parse \ -H "Content-Type: application/json" \ -d '{ "parseURL": "https://example.com", "selector": "h1", "method": "text" }'
